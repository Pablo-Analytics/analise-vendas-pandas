# 📊 Análise de Vendas com Python e Pandas

## 📌 Objetivo
Realizar uma análise exploratória de dados de vendas para identificar padrões e gerar insights relevantes para o negócio.

---

## 🧠 Perguntas de Negócio

- Qual cidade apresenta o maior volume de vendas?
- Qual produto possui a maior média de vendas?
- Qual período (mês) teve melhor desempenho?
- Quais registros representam alta performance (vendas > 300)?
- Qual canal de vendas é mais eficiente?

---

## 🛠 Tecnologias Utilizadas

- Python
- Pandas

---

## 📂 Estrutura dos Dados

- data  
- cidade  
- produto  
- canal  
- vendas  

---

## 📈 Principais Análises

- Uso de `groupby` para agregações
- Cálculo de soma e média
- Ordenação de dados
- Filtros condicionais
- Identificação de padrões de negócio
